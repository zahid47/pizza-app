## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/en)

### Environment Variables

see `.env.test` file for all the env variables you need to set.

### Run Locally

Clone the project

```bash
  git clone https://github.com/zahid47/pizza-app.git
```

Install dependencies

```bash
  yarn

  cd client

  yarn

  cd ../
```

Start server and client

```bash
  yarn all
```

Run Tests

```bash
  yarn test
```
